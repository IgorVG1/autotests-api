## Основные методы моделей

- **modeldump()** — превращает модель в обычный словарь данных Python.
- **modeldumpjson()** — превращает модель в JSON-строку.
- **modelvalidate()** — валидирует входящие данные и создаёт модель, если всё ок.
- **modelvalidatejson()** — валидирует данные сразу из JSON-строки.
- **modelvalidatestrings()** — валидирует объект, где все поля — строки.
- **modelcopy()** — создаёт копию объекта модели (можно изменить какие-то значения).
- **modelconstruct()** — быстро собирает объект из уже проверенных (доверенных) данных без валидации (редко используется новичками).
- **modeljsonschema()** — генерирует схему (описание структуры модели и типов) для внешних систем.
- **modelextra** — возвращает все дополнительные поля, которые не определены явно в модели (если разрешено в настройках).
- **modelfieldsset** — возвращает имена полей, которые были явно указаны при создании объекта.

---

## Вспомогательные методы

- **modelrebuild()** — перестраивает внутреннюю схему модели (нужно редко, только для продвинутых сценариев с вложенными моделями).
- **modelpostinit()** — вызывается после создания модели, можно переопределить для дополнительной инициализации.
- **modelparametrizedname()** — вычисляет имя класса для обобщённых (generic) моделей (обычно не используется новичками).

***

## Настройки (Config, class variables)

- **ConfigDict**: Это словарь с настройками для всей модели, например:
    - `extra`: что делать с лишними полями (ignore — игнорировать, allow — добавлять, forbid — выдавать ошибку).
    - `fromattributes`: брать ли данные из атрибутов, а не только из словаря.
- **privateattributes** — описание приватных (служебных) полей.
- **signature** — объект подписи конструктора модели (для продвинутого использования).
- **pydanticfields** — словарь со всеми полями модели и их параметрами.
- **pydanticextra** — дополнительная информация, если разрешено лишнее.
- **pydanticfieldsset** — имена явно заданных при создании полей.
- **pydanticprivate** — приватные (служебные) атрибуты у конкретного объекта модели.
- **modelfields / modelcomputedfields** — все обычные и вычисляемые поля модели.

***

## Пример использования методов

```python
user = UserModel(name="Anna")
user_dict = user.modeldump()    # {'name': 'Anna'}
user_json = user.modeldumpjson() # '{"name":"Anna"}'
clone = user.modelcopy(update={"name": "Ivan"})
```


***

## Важно: устаревшие методы

- Вместо `dict()` и `json()` — используем теперь `modeldump()` и `modeldumpjson()`.
- Вместо `validate()` — `modelvalidate()`.
- Вместо `schema()` — `modeljsonschema()`.
