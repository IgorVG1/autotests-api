# Allure


## Генерация результатов выполнения автотестов (*allure-results*)

> python -m pytest --alluredir ./allure-results

- Опция **--alluredir=./allure-results** указывает, что результаты выполнения тестов будут сохранены в папку ./allure-results, которая затем используется для генерации отчета.


## Генерация отчета (*allure-report*)


### Генерация статического отчета

> allure generate ./allure-results --output=./allure-report

- При успешном выполнении команда создаст папку allure-report с содержимым, необходимым для генерации html-страницы отчета: **index.html**
- Файл **index.html** можно открыть в браузере, чтобы увидеть сгенерированный отчет
- Либо можно запустить локальный сервер для просмотра отчета:

> allure open ./allure-report


### Генерация динамического отчета

> allure serve ./allure-results

- Эта команда:
1. Генерирует временный отчет
2. Запускает локальный сервер
3. Автоматически открывает отчет в браузере

- Чтобы остановить сервер, нужно нажать **Ctrl+C** в консоли
